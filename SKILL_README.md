# OpenClaw Killer Skill

An ironic OpenClaw Skill — let OpenClaw uninstall itself 🦞

## Installation

```bash
# Clone the repo
git clone https://github.com/orange2ai/openclaw-killer.git
cd openclaw-killer

# Install the Skill
./install-skill.sh
```
